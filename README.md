## Design system

**[`DESIGN.md`](./DESIGN.md) is the source of truth for the site's visual language** — colors, typography, spacing, buttons, components, and accessibility rules. Every new page, component, or admin screen must follow it. Design tokens live as CSS custom properties in [`src/globals.css`](./src/globals.css) (`:root`); always reference the token, never a hard-coded value.
